# Image Slider em JavaScript

Projeto front-end desenvolvido com **JavaScript puro**, focado na criação de um **slider de imagens interativo**, com múltiplas formas de navegação e controle de estado da interface.

O objetivo deste projeto é demonstrar domínio de **manipulação do DOM**, controle de eventos e construção de componentes reutilizáveis de UI.

---

## Funcionalidades

- **Slider horizontal de imagens**, com imagem principal centralizada na tela
- **Navegação por botões (<- / ->)**
- **Navegação via scroll horizontal**
- **Menu de miniaturas (thumbnails)** que permite selecionar diretamente qualquer imagem
- Sincronização entre:
  - imagem ativa no slider
  - miniatura selecionada
- Interface simples e focada na experiência do usuário

---

## Tecnologias Utilizadas

- JavaScript (ES6+)
- HTML5
- CSS3
- Manipulação direta do DOM
- Eventos de clique e scroll
- Organização de código orientada a componentes

---

## Objetivo do Projeto

Este projeto foi desenvolvido como um **exercício prático de JavaScript**, com foco em:

- Criação de componentes interativos
- Controle de estado da interface
- Navegação fluida e intuitiva
- Escrita de código limpo, organizado e reutilizável

Ele também faz parte do meu **portfólio de front-end**, sendo um projeto pequeno e isolado que demonstra conceitos fundamentais utilizados em aplicações reais.
